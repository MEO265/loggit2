# Set echo

Set echo

## Usage

``` r
set_echo(echo = TRUE, confirm = TRUE)
```

## Arguments

- echo:

  Should log messages be echoed to `stdout`?

- confirm:

  Print confirmation message of echo setting?

## Value

Invisible the previous echo setting.

## Examples

``` r
if (FALSE) { # \dontrun{
 set_echo(TRUE)
 set_echo(FALSE)
} # }
```
